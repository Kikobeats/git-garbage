# git-garbage

<p align="center">
  <br>
  <img src="demo.png" alt="git-garbage">
  <br>
</p>

> It deletes merged/squashed local git branches after deleting them on the remote repository.

## Install

```bash
npm install --global git-garbage
```

## Usage

```sh
git-garbage
```
